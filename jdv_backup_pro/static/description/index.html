Easy Installation

  1. Install the module from Apps menu
  2. Go to Database Backups menu
  3. Create a new backup configuration
  4. Choose storage type and enter credentials
  5. Set your schedule and retention policy
  6. Click "Test Connection" to verify
  7. Done! Your data is now protected

Optional Dependencies

Install only what you need:

  • paramiko - For SFTP support: pip install paramiko
  • boto3 - For Amazon S3: pip install boto3
  • google-api-python-client - For Google Drive: pip install google-api-python-client google-auth
  • cryptography - For encryption: pip install cryptography

Frequently Asked Questions

Does this module work with Odoo 19 Community?

Yes! This backup module is fully compatible with both Odoo 19 Community and Enterprise editions.

How do I set up Amazon S3?

Create an S3 bucket and IAM user in AWS console. Enter Access Key, Secret Key, bucket name and region in the module settings. Works with any S3-compatible service (DigitalOcean Spaces, MinIO, Wasabi).

How do I set up Google Drive?

Create a Service Account in Google Cloud Console, download the JSON key file, and paste its content in the module settings. Share your destination folder with the service account email.

Is the encryption secure?

Yes! We use AES-256 encryption with PBKDF2 key derivation (100,000 iterations). This is military-grade encryption suitable for sensitive data.

Can I backup to multiple destinations?

Yes! Enable "Also Save Locally" to keep a local copy in addition to your cloud storage.
